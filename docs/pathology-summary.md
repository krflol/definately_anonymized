# Pathology Summary

## Specimen

Patient-right cervical-chain lymph-node conglomerate, needle core biopsy.

## Diagnosis

Metastatic squamous cell carcinoma, moderate to poorly differentiated.

## Immunohistochemistry

- CK5: positive.
- p16: positive with block staining.

## Flow Cytometry

No significant lymphoid immunophenotypic abnormalities were reported.

## Interpretation

The biopsy confirms squamous cell carcinoma in a right cervical lymph node. p16 block positivity raises concern for HPV-mediated carcinoma, commonly from an oropharyngeal primary. The report recommended clinical and radiologic correlation to identify the primary site.

Flow cytometry makes lymphoma less likely as the cause of the biopsied mass.

## Missing Or Pending Details

- Primary tumor site.
- Full stage.
- HPV DNA/RNA or genotype, if ordered.
- PD-L1 CPS, if ordered.
- Tumor-board recommendations.
