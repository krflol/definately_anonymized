# Clinical Trial Search Playbook

## Search Terms

- `HPV positive oropharyngeal squamous cell carcinoma`
- `p16 positive oropharyngeal cancer`
- `HPV16 head and neck squamous cell carcinoma`
- `metastatic squamous neck cancer occult primary`
- `BNT113 pembrolizumab`
- `PRGN-2009 HPV`
- `PDS0101 pembrolizumab HPV16`
- `VB10.16 pembrolizumab`
- `E7 TCR-T HPV`
- `HPV ctDNA NavDx oropharyngeal`
- `deintensification HPV oropharyngeal`

## Eligibility Details To Collect

- Exact diagnosis and stage.
- Primary site.
- HPV genotype.
- PD-L1 CPS.
- Prior treatment status.
- HLA type if TCR-T is being considered.
- ECOG performance status.
- Organ function labs.
- Hearing status if cisplatin is possible.
- Autoimmune disease, transplant history, chronic steroids, or immunosuppression.

## Practical Workflow

1. Finish staging first.
2. Ask whether treatment intent is curative.
3. Ask whether any trial is appropriate before standard treatment begins.
4. If yes, ask for referral immediately because screening can take time.
5. If no, repeat the trial search after major decision points: PET/CT, surgery if performed, final pathology, radiation/chemoradiation, and surveillance.
